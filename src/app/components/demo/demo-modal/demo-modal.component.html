modal
Both have template modals which are useless, but both have capabilities of showing a modal on the fly and receiving an async response. ngmodal takes a size option, but can just put that in the modal template, so not needed. Pushed boilerplate code to a modal base class which simplifies things considerably for ng2modal, but little for ngmodal. There was a lot of work required to get ng2modal up and running like a component. Once done I guess they're somewhat equivalent, but ngmodal's service to open a component modal is by far superior.
